
## Description

Boss Dashboard

## Installation

```bash
$ npm install
```

## Configuration

```dotenv
PORT=3001
DATABASE_HOST=localhost
DATABASE_NAME=dbname
DATABASE_USER=username
DATABASE_PASSWORD=password
DATABASE_PORT=5432
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
